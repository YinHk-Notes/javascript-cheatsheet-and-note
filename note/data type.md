## Data type

- **Primitive data type:** 

  a data type that has a primitive value. A primitive value is a value that has no properties or methods.
  JavaScript defines 5 types of primitive data types:

- `string`
- `number`
- `boolean`
- `null`
- `undefined`

- **Non-primitive data type**
  
  It is reference type.
  - `Object`
  - `Array`
  - `Function`
  - `Date`
  - `Regx`
